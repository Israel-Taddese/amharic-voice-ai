# Product Plan: AmharicVoice AI

## Vision

Build a simple, trustworthy Amharic ↔ English voice translator for Ethiopian users, diaspora communities, travelers, students, small businesses, and community organizations.

## MVP user story

As a user, I can speak in Amharic or English, see the transcript, see the translation, and play the translated audio.

## MVP features

- Amharic → English voice translation
- English → Amharic voice translation
- Transcript preview
- Text translation result
- Audio playback of translated text
- Privacy-first backend design

## Differentiation

Most translation apps treat Amharic as an extra language. This app should feel Amharic-first: clear Amharic UI, Ethiopian context, diaspora use cases, low-data design, and future support for common phrases used in family, travel, church, school, healthcare, and business contexts.

## Roadmap

### Phase 1
- Local backend
- iOS prototype
- Azure STT/Translator/TTS integration

### Phase 2

### Phase 3

### Phase 4

- Web app/PWA
